# 目標とする機能
## 概要

1. プラグインモードで起動
1. シャッターボタンで静止画を撮影する
1. プラグインモードに入ってから撮影した画像を記憶
1. シャッターボタン長押しで、画像をLineアカウントに送信
   * 送り先Lineアカウントは、ひとまず固定
